* # Run local server :-
   * *To run the server provided by the django :* 
        1. Open cmd.
        2. Make sure that you changed directory to the  stockMarket folder.
        3. Use this command *"python manage.py runserver"*.
        4. Copy the ip address from cmd to your browser.
